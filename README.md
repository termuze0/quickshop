# QuickShop

A simple eCommerce web application built with **Next.js**, **TypeScript**, **Tailwind CSS**, **Shadcn UI**, and **Redux Toolkit**.  


## Features

- Product listing page with cards showing:
  - Title, price, rating, category, and thumbnail image  
  - "Add to Favorite" button for users (Redux-powered)  
- Infinite scroll pagination (`?limit=10&skip=10`)  
- Search bar to filter products by title  
- Responsive grid layout using **Tailwind CSS** and **Shadcn UI**  
- Dark/Light mode support in Navbar (Shadcn UI)  
- Redux Toolkit state management for favorites and products  
- TypeScript for type safety  

---

## Tech Stack

- **Framework:** Next.js 16 (App Router)  
- **Language:** TypeScript  
- **Styling:** Tailwind CSS + Shadcn UI  
- **State Management:** Redux Toolkit + React-Redux  
- **API:** DummyJSON via Axios  
- **Icons:** Lucide React
